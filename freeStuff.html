Free Stuff

One of the many benefits of being a student is free stuff! Here you will find a list of free or discounted things you can get with your OSU email. This list certainly is not comprehensive, but has an emphasis on tools useful for developers. If you think anything should be added to this list then please contact us!

Microsoft Imagine

This is a great place to find a lot of different Microsoft products for free!

JetBrains Professional Developer Tools

JetBrains offers IDEs built for specific languages such as Java, PHP, Ruby, Python, JavaScript, .NET, and Objective-C. These are free while a student!

GitHub Student Developer Pack

Tons of different tools packed into this one offer. Hosting/domain services, private GitHub repositories, credit for Stripe transactions, Unreal Engine, etc.

50% off Amazon Prime

You never realize how much you want Amazon Prime until you don't have it anymore. Really good deal to take advantage of while still in school.
